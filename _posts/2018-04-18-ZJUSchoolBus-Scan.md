这一题是web5端口扫描和目录爆破的题，分两个阶段：
## 端口扫描
这一步要求扫描指定网站的ssh端口，提示给出了用来扫描的工具nmap，全平台适用。
ubuntu下直接apt-get安装就好了：
```
sudo apt-get install nmap
```
完成后直接通过 `nmap -h` 查看用法：
```
Usage: nmap [Scan Type(s)] [Options] {target specification}

  --version-all: Try every single probe (intensity 9)
  --version-trace: Show detailed version scan activity (for debugging)

```
最后有一个EXAMPLES，直接照着来，题目中提到了为了缓解服务器压力，指定了一个端口范围9000-11000。看参数说明`-p`是指定端口范围：
```
nmap -v -p 9000-11000 url
```
很快就返回了发现的端口：
```
PORT      STATE SERVICE
xxxxx/tcp open  unknown

```
第一步完成。
## 目录爆破
目录爆破是根据扫描出来的端口对另一个网站进行目录爆破。这个爆破是一个很暴力的过程，根据字典去遍历，找到网站可访问的目录。这个过程有软件可以实现，题目也给了提示，用DirBuster或者Zap。  
这里选择DirBuster，随意一搜，好多[安装教程](https://www.hackersb.cn/study/34.html)。  
下载链接在[这里](https://sourceforge.net/projects/dirbuster/)。  
安装好指定网址进行目录爆破就好，这个过程是真费时间，可以把线程数开多一点，会快一些(~~玉泉的网实在渣~~)。  
即使多开线程也要好久时间，扫描出现一些waiting的目录就可以手动输入浏览器试试看，很快就会看到一个`\flag`的文件夹，很激动有木有，但是是骗人的。真是有很多奇奇怪怪的目录，和网址一样奇怪。。。多试几个就找到flag了。
